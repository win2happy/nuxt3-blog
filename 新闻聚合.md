# 新闻聚合

## 每天 60 秒读懂世界
GET
https://60s.viki.moe/v2/60s
如果需要 RSS 订阅，可以使用 /v2/60s/rss 端点
encoding 参数支持 text/json/markdown/image/image-proxy 5 种，默认 json
区别：
image: 重定向到图片直链，适合大多数本地调用场景，简单快速
image-proxy: 代理图片，无法访问直链时可以尝试代理模式，适合网站引用
text: 以漂亮格式化的纯文本形式返回，简单、直观
json: 以格式化的 JSON 形式返回，更多信息和数据掌控度
markdown: 针对 AI 和 markdown 场景优化，适合 AI 和 md 渲染场景
请求参数
Query 参数
date
string 
可选
可空，默认今天，格式： 2025-03-01
encoding
string 
可选
编码方式，支持 text/json/markdown/image/image-proxy 5 种
force-update
string 
可选
是否不使用缓存，强制重新拉取数据源仓库最新数据，一般用在「数据源 JSON 数据更新但 API 还是缓存数据，需要更新」的场景

## AI 资讯快报
GET
https://60s.viki.moe/v2/ai-news
提供全面的 AI、大模型 等领域的最新 AI 资讯，数据来源于 https://ai-bot.cn/daily-ai-news
因为新闻是当天实时更新，建议在傍晚 22:00 后调用，或者传入指定日期获取历史数据。
请求参数
Query 参数
date
string 
可选
新闻日期，可空，默认当天，格式： 2025-11-11
all
string 
可选
是否获取所有日期，默认为空，如果需要一次性拉取所有数据，可以设置为 1
encoding
string 
可选
编码方式，支持 text/json/markdown
