# 新闻聚合页面 - 保存图片功能

## 功能说明

### 1. 保存图片按钮
在页面头部有一个"保存图片"按钮，点击后会弹出模态框，让您选择要保存的模块。

### 2. 可保存的模块

- **完整页面** - 保存整个新闻聚合页面
- **60秒读懂世界（红色卡片）** - 生成经典红色背景的新闻卡片（如图所示）
- **60秒读懂世界（当前样式）** - 保存页面当前的新闻列表样式
- **实时热搜** - 保存所有平台的热搜榜单
- **历史上的今天** - 保存历史事件时间轴
- **今日黄历** - 保存黄历宜忌信息
- **每日一语** - 保存励志语录卡片

### 3. 快捷按钮

#### 60秒读懂世界模块
- **红色卡片按钮** - 直接生成经典红色背景卡片
- **刷新数据按钮** - 刷新新闻数据

#### 每日一语模块
- **保存图片按钮** - 保存当前卡片
- **分享按钮** - 分享到其他应用（支持系统分享功能）

### 4. 红色卡片样式

红色卡片会自动生成以下内容：
- 红色渐变背景
- 当前日期（大字体）
- "60秒读懂世界"标题
- 星期和农历日期
- 所有新闻列表（带序号）
- 优雅的圆角和阴影效果

### 5. 使用方法

1. 点击页面顶部的"保存图片"按钮
2. 在弹出的模态框中选择要保存的模块
3. 等待图片生成（会显示加载动画）
4. 图片会自动下载到您的设备

### 6. 分享功能

对于支持的组件（如每日一语），点击分享按钮可以：
- 如果浏览器支持 Web Share API，会打开系统分享菜单
- 否则会自动下载图片

### 7. 技术实现

- 使用 `html2canvas` 库将 DOM 元素转换为图片
- 高清输出（2-3倍缩放）
- 支持透明背景和自定义背景色
- 自动处理跨域图片
